### about

Engenheira de software com raízes em física, matemática e filosofia. O que me interessa não é usar ferramentas — é entender por que elas funcionam ou falham no nível mais fundamental possível.

Trabalho na camada onde hardware e software deixam de ser distintos: sistemas embarcados, infraestrutura de IA, computação de baixo nível.

Estudo física quântica não como curiosidade acadêmica — como base para o próximo paradigma computacional.

A filosofia não é separada disso. É o que evita que o pensamento técnico se torne mecânico.

<!-- PHILOSOPHY -->
### `>` operating principle

```
Uso Python como linguagem de orquestração científica e C++ onde performance
e controle de memória não são negociáveis. Rust quando o sistema precisa ser
seguro por construção, não por convenção. CUDA para computação em GPU —
porque saber o que acontece abaixo da API é o que separa quem usa de quem
constrói. Estudo Haskell não para usar em produção — para pensar com precisão
sobre o que um programa realmente faz.

A direção: física computacional, sistemas inteligentes embarcados e
infraestrutura de IA — onde o mundo físico e o computacional se tornam
indistinguíveis.
```
